This emulator is a work in progress, and is based on [SnesJs](https://github.com/elzo-d/SnesJs)

# Usage

Clone the repo, create a RedSea `ISO.C` image and add a `ROMs` folder, `#include "Run";`

# Controls

D-PAD: `ARROW KEYS`

Y: `a`

B: `z`

X: `s`

A: `x`

L: `d`

R: `c`

Select: `SHIFT`

Start: `ENTER`
